# hvac1

Marketing site for **Conquer Home Services** (HVAC).

- **Repository:** [github.com/Stonebrook-Digital/hvac1](https://github.com/Stonebrook-Digital/hvac1)

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Production build: `npm run build` then `npm run start`.

## Push to GitHub

If this folder is not yet linked to the remote:

```bash
git remote add origin https://github.com/Stonebrook-Digital/hvac1.git
git branch -M main
git push -u origin main
```

If `origin` already exists, use `git remote set-url origin https://github.com/Stonebrook-Digital/hvac1.git` instead.
